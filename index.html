Box 2D light particle

Box2D light particle is a library intended for add lights to particles with libgdx.

Features:

  • Multiple particles system
  • Apply textures to particle, use HALO texture, use no texture
  • Add lights to particles
  • Add collisions to particles
  • Physic system:
    • Add/Remove physics forces
    • Linear force
    • Radial force
    • Brownian force
  • Add rules to particles and system
  • Randomize forces and rules
